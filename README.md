# <h1 style="color:#ff4444; text-align:center;">Robco</h1>

FullStackAi Platform is a BIM-first, multi-service AI system designed to turn complex project data into structured, traceable, and reproducible outputs.

The platform is built around a centralized control plane, deterministic data systems, and cost-aware AI execution. It integrates BIM ingestion (IFC, PDFs, schedules, and markups), event‑driven workflows, semantic caching, and scoped memory into a single operational backbone.

Rather than treating AI as a feature layer, this system treats it as infrastructure—enforcing strict contracts, preserving source‑of‑truth data, and generating auditable deliverables.

The initial vertical focuses on BIM project intake and analysis, producing structured summaries, issue registers, and versioned deliverables from real project data.

This repository is structured as a modular monorepo so services can scale, separate, and evolve independently without breaking system integrity.

*Start with Docker Compose and disciplined service boundaries.*
